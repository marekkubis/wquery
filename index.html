About WQuery

WQuery is a domain specific language designed to query WordNet-like lexical databases. WQuery operates on platforms that provide Java Runtime Environment and works with wordnets stored in XML files. It may be used as a standalone application or as an API to a lexical ontology in Java based systems. If you have any questions about the project please contact me by mail.
